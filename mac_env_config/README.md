#mac下环境变量安装配置
## brew 安装
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## node安装
```
$ brew update
$ brew install node
$ node -v
$ npm -v
```
## node 更新
```
npm install -g n  安装n模块 是专门用来管理node.js的版本的
n stable          升级node.js到最新稳定版
n v0.10.26  或 n 0.10.26  安装对应版本node
```
