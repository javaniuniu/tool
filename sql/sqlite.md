## sqlite 安装流程
```
$ wget https://www.sqlite.org/2019/sqlite-autoconf-3280000.tar.gz
$ cd sqlite-autoconf-3280000
$ tar -zvxf  sqlite-autoconf-3190300.tar.gz
$ ./configure
$ make
$ sudo make install
```

## 登录
```
$ sqlite3
```
